# Backend Hotel Popayán

Proyecto desarrollado para la asignatura de **Desarrollo de Aplicaciones Móviles** en la Uniautónoma.

## Tecnologías Utilizadas
* **FastAPI**: Framework web.
* **PostgreSQL**: Base de datos relacional.
* **SQLAlchemy**: ORM para la gestión de datos.
* **JWT (JSON Web Tokens)**: Para la seguridad y autenticación.

## Instalación
1. Clonar el repositorio.
2. Instalar dependencias: `pip install -r requirements.txt`.
3. Ejecutar el servidor: `uvicorn app.main:app --reload`.
